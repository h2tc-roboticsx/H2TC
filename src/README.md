# Source code

* **recorder.py**: the master recording script
* **postprocess.py**: processing the raw data
* **annotate.py**: annotation tool
* **log.py**: generate instructions and maintain logbook
* ***utils/***: utility modules for the recording, processing of each hardware
* ***tools/***: some auxiliary tools
* ***natnet_client/***: NatNet client to receive OptiTrack data from OptiTrack server
* ***remote_display/***: display instructions on the monitors
