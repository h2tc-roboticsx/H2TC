# Utility

* **zed.py**: utility functions of ZED camera
* **event.py**: utility functions of Event camera
* **he.py**: utility functions of processing [Hand Engine](https://stretchsense.com/solution/hand-engine/) data
* **optitrack.py**: utility functions of processing OptiTrack data
* **plot_motion.py**: plot the hand motion. 
* **align.py**: align streams
